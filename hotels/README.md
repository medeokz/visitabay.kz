# Управление гостиницами (Express + MySQL)

## Требования

- Node.js 16+
- MySQL (например XAMPP)

## Установка

1. Скопировать `.env.example` в `.env` и при необходимости изменить настройки БД и пароль админа:
   ```bash
   copy .env.example .env
   ```

2. Создать базу и таблицы (вручную или через phpMyAdmin):
   - Выполнить `db/schema.sql` в MySQL, либо при первом запуске таблицы создадутся автоматически.

3. Установить зависимости и запустить:
   ```bash
   npm install
   npm start
   ```

## Адреса

- **Сайт (список гостиниц):** http://localhost:3000/
- **Карточка гостиницы:** http://localhost:3000/hotel/:id
- **Админ-панель:** http://localhost:3000/admin
- **Вход (админ и хозяева):** http://localhost:3000/admin/login
- **Кабинет хозяина гостиницы:** http://localhost:3000/owner
- **Управление хозяевами (только админ):** http://localhost:3000/admin/owners

Роли: **admin** — полный доступ ко всем гостиницам и создание учётных записей хозяев; **owner** — только свои гостиницы (те, у которых `owner_id` = его id). Админ создаёт хозяев в разделе «Хозяева», затем хозяин входит тем же логином и попадает в /owner.

## API

- `GET /api/hotels` — список гостиниц (публично)
- `GET /api/hotels/:id` — одна гостиница (публично)
- `POST /api/hotels` — создать (админ или хозяин; у хозяина гостиница привязывается к нему)
- `PUT /api/hotels/:id` — обновить (админ — любая, хозяин — только своя)
- `DELETE /api/hotels/:id` — удалить (админ — любая, хозяин — только своя)
